`ORI-A.py` is een Python library voor het maken, controleren en validaren van ORI-A XML. **WIP.**

Voor meer informatie over ORI-A, zie <https://ori-a.nl>.

# Installatie

Voor nu:

``` shell
git clone https://github.com/Regionaal-Archief-Rivierenland/ori-a.py
sudo pip install . # of laat sudo weg als je op Windows zit/weet wat je doet
```
